## Fredbear and Friends: Revelation
The source code repo for [Fredbear and Friends: Revelation](https://gamejolt.com/games/faf-rev/754442).
Includes the C++ source code and some blueprint assets such as data assets and other logic blueprints as well. Non blueprint assets such as static meshes, textures, etc. not included due to copyright and file size.

##### **Note:** Dependencies for developers listed in the [uproject](FaF_Revelation.uproject) and [ToroUtilities' uplugin](https://github.com/RedCraft86/ToroUtilities/blob/v3.0-SP/ToroUtilities.uplugin) files.
